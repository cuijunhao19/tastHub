# Vue + Vite 任务清单小项目说明

## 一、项目介绍

基于 Vue3 + Vite 构建的轻量级任务清单应用，实现任务的基础管理能力，界面简洁、交互友好，支持本地数据持久化，满足日常任务记录与管理需求。

## 二、核心技术栈

- 框架：Vue3（Composition API）
- 构建工具：Vite
- 数据持久化：LocalStorage
- 样式：Scoped CSS（可选：UnoCSS/Tailwind CSS）

## 三、快速开始

1. 安装依赖

bash

运行

```
npm install
```

2. 启动开发服务

bash

运行

```
npm run dev
```

3. 打包构建

bash

运行

```
npm run build
```

## 四、核心功能

- ✅ 新增任务：输入任务内容并提交，快速添加待办
- ✅ 任务状态切换：勾选 / 取消勾选，标记任务完成 / 未完成
- ✅ 删除任务：移除单条不需要的任务
- ✅ 清空已完成：一键清除所有已完成的任务
- ✅ 本地持久化：任务数据存储在 LocalStorage，刷新 / 重启页面不丢失

## 五、核心目录结构

plaintext

```
src/
├── components/       # 组件目录
│   └── TaskItem.vue  # 单个任务项组件
├── App.vue           # 根组件（核心逻辑：任务增删改查）
└── main.js           # 项目入口文件
```

## 备注

- 项目无后端依赖，所有数据仅存储在浏览器本地，清空浏览器缓存会导致数据丢失；
- 适配主流现代浏览器，基于 Vue3 语法开发，无兼容 Vue2 相关代码。
